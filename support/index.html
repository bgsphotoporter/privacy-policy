자주 묻는 질문

사진 전체 접근 권한이 필요한 이유는 무엇인가요?

사진과 영상 개수를 계산하고, 백업할 원본 리소스를 읽고, 폴더에서 가져올 때 기존 사진 라이브러리와 중복을 비교하기 위해 필요합니다.

사진이나 영상이 개발자 서버로 업로드되나요?

아니요. BGS Photo Porter는 사용자가 선택한 백업과 가져오기 작업을 기기 안에서 처리합니다. 앱 자체 서버로 사진이나 영상 파일을 업로드하지 않습니다.

백업 파일은 어디에 저장되나요?

사용자가 Files 앱에서 선택한 폴더에 저장됩니다. 앱은 촬영 날짜 기준으로 연/월/일 폴더를 만들어 사진과 영상을 정리합니다.

Folder to Photos는 무엇인가요?

외부 폴더의 사진과 영상을 iPhone 또는 iPad의 사진 앱으로 가져오는 기능입니다. 가져오기 전에 가능한 범위에서 중복 항목을 확인합니다.

히스토리는 어디에 저장되나요?

백업과 가져오기 기록은 기기 내 앱 Documents 영역에 저장됩니다. 히스토리 화면에서 선택 삭제 또는 전체 삭제를 할 수 있습니다.

광고나 후원 결제가 앱 기능을 제한하나요?

아니요. Coffee or Cake 항목은 선택형 개발자 후원용 인앱결제이며, 앱의 핵심 백업과 가져오기 기능을 잠그지 않습니다.
